# Spectacles AI Context

A curated collection of Spectacles development resources optimized for AI-assisted development.

## Purpose

This repository serves as a comprehensive context source for AI systems, enabling:
- **AI-powered code assistance** for Spectacles/Lens Studio development
- **Context-aware development suggestions** based on official examples

## What's Included

### Documentation (`docs/`)
Complete technical documentation covering:
- Spectacles APIs and features
- Best practices and design guidelines
- Framework references
- Getting started guides

### Unpacked Packages (`packages/`)
Reusable asset packages and utilities:
- Authentication and authorization tools
- Camera and image processing
- Input helpers and UI components
- Spatial computing utilities
- And many more...

### Samples (`samples/`)
Complete working examples demonstrating:
- AI and ML integrations
- Connected Lens experiences
- Spatial persistence
- Hardware integrations (BLE, controllers)
- And many more...

## Usage

This repository is designed to be consumed by AI systems for context-aware assistance. The structured organization allows for efficient retrieval of relevant code examples, documentation, and best practices.
