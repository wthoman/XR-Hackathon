## Create a Lens Project

When you open Lens Studio for the first time, you will see a Home Page with various distribution targets and templates.

You can begin building Lenses for Spectacles in two ways:

1. Start with the Spectacles Starter Project.
2. Create a new Lens project from scratch.

### Spectacles Starter Project

Open the Spectacles Starter Project directly from Lens Studio Home Page.

<p align="center">
  <img
    src="/img/spectacles/misc/spectacles-starter-project-home-page.png"
    style={{ maxWidth: '600px', width: '100%', borderRadius: '8px' }}
  />
</p>

The Spectacles Starter Project includes the [Spectacles Interaction Kit](../../spectacles-frameworks/spectacles-interaction-kit/get-started.mdx), which facilitates the creation of interactive experiences that align with the Spectacles experience. It provides optimal settings for building Lenses for Spectacles.

### Starting a New Project

To create a new Lens project for Spectacles, locate the **Home** button in the top left corner of Lens Studio and click it to return to the Home Page.

<p align="center">
  <img
    src="/img/spectacles/setup/project-info-settings.png"
    style={{ maxWidth: '600px', width: '100%', borderRadius: '8px' }}
  />
</p>

In the Home Page, select either **Default Project** in the top right corner or **Spectacles > Base Template**.

<p align="center">
  <img
    src="/img/spectacles/setup/project-select.png"
    style={{ maxWidth: '600px', width: '100%', borderRadius: '8px' }}
  />
</p>

Then, in the pop up window, in the **Lens Works on** section, select the checkbox next to Spectacles to enable.

<p align="center">
  <img
    src="/img/spectacles/setup/spectacles-project-info-settings.png"
    style={{ maxWidth: '600px', width: '100%', borderRadius: '8px' }}
  />
</p>

Set your [Preview Panel](../start-building/preview-panel) for Spectacles in the next step.
