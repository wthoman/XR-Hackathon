# Compatibility List

## Overview

Spectacles uses the same Lens Studio API set as Snapchat and Camera Kit. However, some APIs work exclusively on Spectacles and are marked as "wearable only." Additionally, certain existing APIs do not function on the Spectacles (2024) device; refer to the list below for details.

:::note
Many features are still being ported and tested from the Snapchat API to Spectacles. You can try using these features since the Lens Studio workflow is similar. **Features that are consolidated are usually accompanied by spectacles samples or asset**. If a feature isn't listed or supported, you can test it and provide [feedback](https://www.reddit.com/r/Spectacles/) to our team . Visit our [Samples Repository](https://github.com/snapchat/Spectacles-Sample/) for development resources. **Asset Library resource won't be linked in this page, you will need to find them in the Assets Library within Lens Studio**.
:::

### Resources per feature

Here is a detailed list of AR-specific and anticipated features, along with their current level of support on Spectacles and related resources. We are dedicated to continually improving the AR experience on Spectacles, so please stay tuned for future updates. Consider this a chance to explore and experiment with the features that are currently available but also a guide to help you finding the right resources for your development needs.
