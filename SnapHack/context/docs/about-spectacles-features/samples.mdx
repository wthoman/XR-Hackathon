# Samples

A comprehensive collection of templates and examples for building Lens experiences on Spectacles.

:::info Repository Migration
While [https://github.com/Snapchat/Spectacles-Sample](https://github.com/Snapchat/Spectacles-Sample) will continue to reflect the latest changes, **it will be deprecated in the future**. We recommend using our new organization at [https://github.com/specs-devs/samples](https://github.com/specs-devs/samples) for the latest samples, packages, and development resources.
:::

:::note
If you wish to use a feature that is not listed below or want to contribute to our sample repository, leave a [feedback to our team](https://www.reddit.com/r/Spectacles/) or check out our [Samples Repository](https://github.com/specs-devs/samples).
:::
