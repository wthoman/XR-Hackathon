# Camera Module

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

Spectacles offers APIs to retrieve the camera frame – what the user is currently seeing – to better understand and build experiences around the user’s real-world environment.

## Requesting \+ Receiving Frames

The [CameraModule] enables developers to request the camera frame. There are two types of frame requests: camera frame requests and still image requests. Camera frame requests deliver a steady stream of images, albeit at lower quality: this approach is recommended for applications that need video-like consistency in frame delivery. Still image requests, on the other hand, take longer to complete but result in a higher resolution image -- this is the preferred approach for applications that need high quality images for processing.

### Camera Frame Requests

To begin receiving camera frames, construct a [CameraRequest], specifying the exact camera of interest (Spectacles have multiple cameras from which to choose: [Left_Color](https://developers.snap.com/lens-studio/api/lens-scripting/enums/Built-In.CameraModule.CameraId.html#left_color), [Right_Color](https://developers.snap.com/lens-studio/api/lens-scripting/enums/Built-In.CameraModule.CameraId.html#right_color), or [Default_Color](https://developers.snap.com/lens-studio/api/lens-scripting/enums/Built-In.CameraModule.CameraId.html#default_color)) and the desired resolution of the received frames (via imageSmallerDimension, which indicates the size in pixels of the smallest dimension to return).

For example:

<Tabs>
<TabItem value="TypeScript">

```ts
this.createEvent('OnStartEvent').bind(() => {
  const cameraModule: CameraModule = require('LensStudio:CameraModule');
  const cameraRequest = CameraModule.createCameraRequest();
  cameraRequest.cameraId = CameraModule.CameraId.Left_Color;
  const cameraTexture = cameraModule.requestCamera(cameraRequest);
});
```

</TabItem>
<TabItem value="JavaScript">

```js
script.createEvent('OnStartEvent').bind(function () {
  const cameraModule = require('LensStudio:CameraModule');
  const cameraRequest = CameraModule.createCameraRequest();
  cameraRequest.cameraId = CameraModule.CameraId.Left_Color;
  const cameraTexture = cameraModule.requestCamera(cameraRequest);
});
```

</TabItem>
</Tabs>

:::note
createCameraRequest may not be invoked inside onAwake event.
:::

The [requestCamera] method takes a [CameraRequest] and returns a [Texture]. The texture contains the opaque handle that refers to the underlying camera data, and can be used as an input into an MLComponent or uploaded to a remote service via RemoteMediaModule.

To receive a notification each time the frame is updated, use the [CameraTextureProvider] which can be accessed via cameraTexture.control. The [CameraTextureProvider] has an [onNewFrame] event that can be utilized as shown below:

<Tabs>
<TabItem value="TypeScript">

```ts
const onNewFrame = (this.cameraTexture.control as CameraTextureProvider)
  .onNewFrame;
const registration = onNewFrame.add((frame: CameraModule.CameraFrame) => {
  // Process the frame
});

this.createEvent('OnDestroyEvent').bind(() => {
  onNewFrame.remove(registration);
});
```

</TabItem>
<TabItem value="JavaScript">

```js
const onNewFrame = cameraTexture.control.onNewFrame;
const registration = onNewFrame.add(function (frame) {
  // Process the frame
});

script.createEvent('OnDestroyEvent').bind(function () {
  onNewFrame.remove(registration);
});
```

</TabItem>
</Tabs>

### Still Image Requests

To request still image frames, create an [ImageRequest] from the [CameraModule], then invoke [requestImage]. The [requestImage] method is asynchronous and will return an [ImageFrame] when complete, which contains the resulting image in a [Texture], as well as a timestamp if needed.

:::warning
Still image requests are not supported in the Lens Studio editor. Please run the Lens on Spectacles.
:::

<Tabs>
<TabItem value="TypeScript">

```ts
@component
export class StillImageRequest extends BaseScriptComponent {
  @input
  image: Image;

  private cameraModule: CameraModule = require('LensStudio:CameraModule');

  onAwake() {
    this.requestStillImage();
  }

  async requestStillImage() {
    const imageRequest = CameraModule.createImageRequest();

    try {
      const imageFrame: ImageFrame =
        await this.cameraModule.requestImage(imageRequest);

      // E.g, use the texture in some visual
      this.image.mainPass.baseTex = imageFrame.texture;
    } catch (error) {
      print('Still image request failed: ' + error);
    }
  }
}
```

</TabItem>
<TabItem value="JavaScript">

```js
// @input Component.Image image

const cameraModule = require('LensStudio:CameraModule');

async function requestStillImage() {
  const imageRequest = CameraModule.createImageRequest();

  try {
    const imageFrame = await cameraModule.requestImage(imageRequest);

    // E.g, use the texture in some visual
    script.image.mainPass.baseTex = imageFrame.texture;
  } catch (error) {
    print('Still image request failed: ' + error);
  }
}
requestStillImage();
```

</TabItem>
</Tabs>

## Getting Camera Information

The Camera API also provides information about the cameras used on Spectacles in order to understand how the camera transforms 3D space into 2D space. This information is provided by the [DeviceCamera] which is accessible via the [DeviceInfoSystem].

<Tabs>
<TabItem value="TypeScript">

```ts
// Select the camera

this.createEvent('OnStartEvent').bind(() => {
  const camera = global.deviceInfoSystem.getTrackingCameraForId(
    CameraModule.CameraId.Left_Color
  );

  // Retrieve camera properties
  const focalLength = camera.focalLength;
  const principalPoint = camera.principalPoint;
  const resolution = camera.resolution;
  const pose = camera.pose;
});
```

</TabItem>
<TabItem value="JavaScript">

```js

script.createEvent('OnStartEvent').bind(function () {
```

</TabItem>

Additionally, [DeviceCamera] includes project and unproject methods that makes it easier to convert 3D points into 2D points and vice versa.

<TabItem value="TypeScript">

```ts
@component
export class ContinuousCameraFrameExample extends BaseScriptComponent {
  private cameraModule: CameraModule = require('LensStudio:CameraModule');
  private cameraRequest: CameraModule.CameraRequest;
  private cameraTexture: Texture;
  private cameraTextureProvider: CameraTextureProvider;

  @input
  @hint('The image in the scene that will be showing the captured frame.')
  uiImage: Image | undefined;

  onAwake() {
    this.createEvent('OnStartEvent').bind(() => {
      this.cameraRequest = CameraModule.createCameraRequest();
      this.cameraRequest.cameraId = CameraModule.CameraId.Default_Color;

      this.cameraTexture = this.cameraModule.requestCamera(this.cameraRequest);
      this.cameraTextureProvider = this.cameraTexture
        .control as CameraTextureProvider;

      this.cameraTextureProvider.onNewFrame.add((cameraFrame) => {
        if (this.uiImage) {
          this.uiImage.mainPass.baseTex = this.cameraTexture;
        }
      });
    });
  }
}
```

</TabItem>

