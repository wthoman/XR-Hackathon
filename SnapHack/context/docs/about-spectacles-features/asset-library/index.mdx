# Asset Library

Check out the Spectacles section on the Lens Studio Asset Library for Spectacles-centered assets.
You can find the [Asset Library](https://developers.snap.com/lens-studio/assets-pipeline/asset-library/asset-library-overview) in Lens Studio.

:::info Repository Migration
While [https://github.com/Snapchat/Spectacles-Sample](https://github.com/Snapchat/Spectacles-Sample) will continue to reflect the latest changes, **it will be deprecated in the future**. We recommend using our new organization at [https://github.com/specs-devs](https://github.com/specs-devs) for samples ([specs-devs/samples](https://github.com/specs-devs/samples)) and packages ([specs-devs/packages](https://github.com/specs-devs/packages)).
:::

:::note Packages marked with 🔒
Some packages are not available in the public repository but might be part of the [specs-devs/context](https://github.com/specs-devs/context) repo as filtered context material for AI-assisted development.
:::
