# Scripting

Lens Studio provides a [scripting](/lens-studio/features/scripting/script-overview) engine for creating rich interactive experiences. With scripts, your Lenses can respond to touch input, play animation and audio, modify Scene Objects, and more.

<Collapsible title="Is there a way to load textures from a local path instead of defining them as an input?">
  <br />
  You can use `requireAsset` to reference assets such as Textures, Materials, or
  Meshes within your script without needing to use `//@input`. Please follow the
  [guidelines](https://developers.snap.com/lens-studio/features/scripting/script-modules)
  on properly using `requireAsset` by file path.

<Tabs>
<TabItem value="TypeScript">

```ts
let material = requireAsset('../Materials/Dartboard.mat') as Material;
let texture = requireAsset('../Textures/Dartboard.png') as Texture;
let mesh = requireAsset('../Meshes/Dartboard.mesh') as RenderMesh;

this.renderMeshVisual.mainMaterial = material;
this.renderMeshVisual.mainPass.baseTex = texture;
this.renderMeshVisual.mesh = mesh;
```

</TabItem>
<TabItem value="JavaScript">

```js
const material = requireAsset('../Materials/Dartboard.mat');
const texture = requireAsset('../Textures/Dartboard.png');
const mesh = requireAsset('../Meshes/Dartboard.mesh');

script.renderMeshVisual.mainMaterial = material;
script.renderMeshVisual.mainPass.baseTex = texture;
script.renderMeshVisual.mesh = mesh;
```

</TabItem>

<Collapsible title="Is there a way to download an image using Fetch and turn it into a texture?">
  <br />
  Once the image has been fetched, [Base64](https://developers.snap.com/lens-studio/api/lens-scripting/classes/Built-In.Base64.html) can be used to handle encoding and decoding, converting the images into a texture that can be applied to materials.

<Tabs>
<TabItem value="TypeScript">

```ts
@component
export class FetchImageToTexture extends BaseScriptComponent {
  // Using InternetModule to fetch images and turn it into a texture
  @input
  internetModule: InternetModule;

  // Using RenderMeshVisual to display the texture
  @input
  renderMeshVisual: RenderMeshVisual;

  // Method called when the script is awake
  async onAwake() {
    let request = new Request(
      'https://developers.snap.com/img/spectacles/misc/spectacles-2024-hero.png',
      { method: 'GET' }
    );
    let response = await this.internetModule.fetch(request);
    print('Response: ' + response);
    if (response.status === 200) {
      let bytes = await response.bytes();
      // Convert bytes to a base64 string
      let base64 = Base64.encode(bytes);
      // Decode the base64 string to a texture
      this.decodeBase64ToTexture(base64).then(this.displayTexture.bind(this));
    }
  }

  decodeBase64ToTexture(base64: string) {
    // Decode the base64 string to a texture using Base64.decodeTextureAsync
    return new Promise((resolve, reject) => {
      Base64.decodeTextureAsync(base64, resolve, reject);
    });
  }

  displayTexture(texture: Texture) {
    if (this.renderMeshVisual) {
      // Set the texture to the main material of the RenderMeshVisual
      this.renderMeshVisual.mainMaterial.mainPass.baseTex = texture;
    }
  }
}
```

</TabItem>

async function onAwake() {
  const request = new Request(

    decodeBase64ToTexture(base64).then(displayTexture);
  }
}

function decodeBase64ToTexture(base64) {
  // Decode the base64 string to a texture using Base64.decodeTextureAsync
  return new Promise(function (resolve, reject) {
    Base64.decodeTextureAsync(base64, resolve, reject);
  });
}

function displayTexture(texture) {

    // Set the texture to the main material of the RenderMeshVisual
    script.renderMeshVisual.mainMaterial.mainPass.baseTex = texture;
  }
}

// Initialize
onAwake();
