# Snap OS 2.0 Design Kit

The **Snap OS 2.0 Design Kit** is the official Figma component library for designing AR experiences on Spectacles. It provides the foundational design elements used throughout Snap OS, ensuring your Lenses maintain visual consistency with the platform.

## What's Included

The design kit contains everything you need to create polished AR interfaces:

- **Design Tokens** - Colors, typography, spacing, and other foundational values optimized for the Spectacles display
- **UI Styles** - Pre-configured text styles, effects, and color palettes for the additive display
- **Components** - Buttons, cards, menus, and other interactive UI elements matching Snap OS patterns
- **Reference Screens** - Example layouts and flows to guide your design decisions

## How to Use

1. **Duplicate the file** - Open the Figma link and duplicate the file to your own Figma workspace
2. **Explore the components** - Browse through the available tokens, styles, and components
3. **Design your interface** - Use the provided elements to mock up your Lens UI
4. **Implement in Lens Studio** - Use [Spectacles UI Kit](../../spectacles-frameworks/spectacles-ui-kit/get-started.mdx) to bring your designs to life

## Design for Spectacles Display

When using the design kit, keep in mind the unique characteristics of the Spectacles display:

- **Additive color space** - White is the boldest color; black appears transparent. See [Color & Material](./ui-design.mdx#color--material) for details.
- **Optimal viewing distance** - Content is most comfortable at approximately 1 meter. See [Positioning & Sizing Content](./positioning-sizing-content.mdx) for guidelines.
- **Portrait aspect ratio** - The display uses approximately 3:4 proportions.
